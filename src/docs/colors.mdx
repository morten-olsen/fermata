## Rules

<div style={{ padding: 24, backgroundColor: '#141416', borderRadius: 12, margin: '24px 0' }}>
  <ol style={{ paddingLeft: 20, lineHeight: 2, color: '#9898A3', fontSize: 14 }}>
    <li><strong style={{ color: '#E8E8ED' }}>Three surfaces, no more.</strong> bg → surface → elevated. If you need a fourth, the layout is too nested.</li>
    <li><strong style={{ color: '#E8E8ED' }}>No pure white or pure black.</strong> Text is #E8E8ED (slightly warm). Background is #0A0A0B (not #000).</li>
    <li><strong style={{ color: '#E8E8ED' }}>Accent is rare.</strong> Currently playing track, tappable names, progress bar, active controls. Never decoration.</li>
    <li><strong style={{ color: '#E8E8ED' }}>Borders are a last resort.</strong> Prefer spacing and surface color shifts.</li>
    <li><strong style={{ color: '#E8E8ED' }}>Album art sets the mood.</strong> Future: Now Playing tints with dominant artwork color.</li>
  </ol>
</div>
