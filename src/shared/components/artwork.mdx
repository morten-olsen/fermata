# Artwork

<p style={{ color: '#9898A3', fontSize: 16, marginBottom: 24 }}>
  The foundational image primitive — renders album art, podcast covers, or book covers with consistent sizing, fallbacks, and optional badges.
</p>

## Design rationale

Fermata's first principle is **"the music is the interface"** — and artwork is how that manifests visually. This component standardizes how artwork appears across every context: cards, grids, detail headers, search results, and the mini player.

## Usage

```tsx
// In a card
<Artwork uri={artworkUrl} size="md" />

// In a detail header
<Artwork uri={artworkUrl} size="xl" heroTransition />

// Audiobook card
<Artwork uri={artworkUrl} aspect="portrait" size="lg" fallbackIcon="book" />

// With download badge
<Artwork uri={artworkUrl} size="md" badge={<DownloadBadge />} />
```
