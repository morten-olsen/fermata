## Design rationale

Every detail screen in Fermata follows the same visual rhythm: large artwork at the top, title and metadata below, action buttons, then a list. This component codifies that pattern while allowing each content type to configure its own shape, metadata, and actions.

## Content types

### Music album
<Canvas of={DetailHeaderStories.Album} />
Square artwork (256px), tappable artist, Play + Shuffle buttons.

### Podcast show
<Canvas of={DetailHeaderStories.PodcastShow} />
Square artwork (192px), episode count in meta, no action buttons.

### Audiobook
<Canvas of={DetailHeaderStories.Audiobook} />
Portrait artwork (3:4 ratio), progress in meta, Continue button.
