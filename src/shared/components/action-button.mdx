## Design rationale

Action buttons are the gateway to listening. They sit below the album/show/book metadata and represent the most important decision on the screen: *start playing*.

## Layout anatomy

```
┌─────────────────┐ ┌─────────────────┐
│  ▶  Play        │ │  ⤮  Shuffle     │
└─────────────────┘ └─────────────────┘
   primary (white)    secondary (elevated)
   flex-1, py-3       flex-1, py-3
   rounded-xl         rounded-xl
```

Both buttons use `flex-1` inside a `flex-row gap-3` container, making them equal width.
