## Design rationale

Empty states are moments where the app could feel broken — or could feel intentionally calm. Fermata chooses calm.

## Variants

<Canvas of={EmptyStateStories.WithoutSubtitle} />

The subtitle is optional. Without it, the empty state is even more minimal — just an icon and a short message.

## Usage

```tsx
<EmptyState
  icon="musical-notes-outline"
  title="No tracks found"
  subtitle="Add a source to start listening"
/>
```

Use Ionicons icon names. Choose icons that suggest what's missing: `search-outline` for empty search, `musical-notes-outline` for no tracks, `disc-outline` for no albums.
