## Design rationale

The most distinctive visual choice in Fermata's content-type system. Audiobooks use **portrait (3:4) artwork** because books are tall — and that shape immediately communicates "this is a book, not an album" when scanning the grid.

## How it differs from AlbumCard

| | AlbumCard | BookCard |
|---|---|---|
| **Aspect ratio** | Square (1:1) | Portrait (3:4) |
| **Grid columns** | 2 | 3 |
| **Badge** | Download indicator | Progress percentage |
| **Fallback icon** | disc | book |
| **Feel** | Record collection | Bookshelf |
