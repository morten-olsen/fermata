## Design rationale

The equalizer bars replace the track number in list rows when a track is playing. They're a subtle but recognizable indicator — you can scan a track list and instantly see what's playing without reading text.

## Where it's used

- **Track rows** — replaces the track number column (size 16)
- **Queue sheet** — replaces the position number for the current track (size 14)

The component pairs with accent-colored text: when bars appear, the track title also turns `text-fermata-accent`.

## Usage

```tsx
{isPlaying ? (
  <EqualizerBars size={16} color={colors.accent} />
) : (
  <Text style={{ color: colors.muted }}>{trackNumber}</Text>
)}
```
