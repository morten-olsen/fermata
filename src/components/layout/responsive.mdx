## Overview

NativeWind handles responsive styling via classes (`sm:px-6 md:px-8`), but sometimes you need a responsive value in JavaScript — for example, setting a `numColumns` prop on `FlatList` or computing padding for `contentContainerStyle`. The responsive hooks provide exactly that.

All hooks share the same breakpoint thresholds as Tailwind, so class-based and hook-based responsive logic stay in sync.

## Hooks

### `useBreakpoint()`

Returns the active breakpoint name based on the current window width.

<Canvas of={ResponsiveStories.Breakpoint} />

```tsx
import { useBreakpoint } from "@/src/components/layout/responsive";

function MyComponent() {
  const breakpoint = useBreakpoint();
  // "base" | "sm" | "md" | "lg" | "xl"
}
```

### `useColumns(config)`

Convenience hook for responsive column counts. Pass a `BreakpointConfig<number>` and get back the resolved column count.

<Canvas of={ResponsiveStories.Columns} />

```tsx
import { useColumns } from "@/src/components/layout/responsive";

function AlbumGrid() {
  const columns = useColumns({ base: 2, sm: 3, md: 4, lg: 5 });

  return (
    <FlatList
      data={albums}
      numColumns={columns}
      key={columns} // re-mount when columns change
      renderItem={renderAlbum}
    />
  );
}
```

### `useResponsiveValue(config)`

General-purpose hook that resolves a value from a breakpoint config map. Falls back to the nearest smaller breakpoint that has a defined value.

<Canvas of={ResponsiveStories.ResponsiveValue} />

```tsx
import { useResponsiveValue } from "@/src/components/layout/responsive";

function DetailScreen() {
  const padding = useResponsiveValue({ base: 16, md: 24, lg: 32 });

  return (
    <FlatList
      contentContainerStyle={{ padding }}
      // ...
    />
  );
}
```

## NativeWind class-based responsive

For styling that lives in `className`, use NativeWind's built-in responsive prefixes directly. The breakpoints are identical.

```tsx
<View className="px-4 sm:px-6 md:px-8 lg:px-12">
  <Text className="text-sm md:text-base lg:text-lg">
    Responsive text
  </Text>
</View>
```

Use the hooks only when you need a responsive value in JavaScript (e.g., `numColumns`, `contentContainerStyle`, conditional rendering). Prefer NativeWind classes for everything else.
