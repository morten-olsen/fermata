## Design rationale

Bottom sheets are one of only two modal patterns in Fermata (the other is the full-screen Now Playing overlay). They're used sparingly — only for track actions and the playback queue.

## Original API (children)

The simplest way to use BottomSheet — pass any content as children:

```tsx
<BottomSheet visible={showSheet} onDismiss={() => setShowSheet(false)}>
  <View style={{ paddingHorizontal: 20 }}>
    {/* Any content */}
  </View>
</BottomSheet>
```

The sheet handles its own animation lifecycle — just toggle `visible`. Safe area padding is applied automatically at the bottom.

## Compound API

<p style={{ color: '#9898A3', fontSize: 14, marginBottom: 16 }}>
  For structured action sheets, use the compound sub-components. They provide consistent spacing, icon alignment, and a destructive variant out of the box.
</p>

<Canvas of={BottomSheetStories.WithCompoundAPI} />

```tsx
<BottomSheet visible={showSheet} onDismiss={() => setShowSheet(false)}>
  <BottomSheet.Header title="Track Actions" />
  <BottomSheet.Item icon="heart-outline" label="Favourite" onPress={handleFavourite} />
  <BottomSheet.Item icon="list-outline" label="Add to Mix Tape" onPress={handleAddToMixTape} />
  <BottomSheet.Divider />
  <BottomSheet.Item icon="disc-outline" label="Go to Album" onPress={handleGoToAlbum} />
  <BottomSheet.Item icon="person-outline" label="Go to Artist" onPress={handleGoToArtist} />
</BottomSheet>
```

You can mix children and compound sub-components — the root just renders whatever is inside.

## Action sheet pattern

A common pattern for confirmations — a header, a safe action, a divider, then a destructive action:

<Canvas of={BottomSheetStories.ActionSheet} />

```tsx
<BottomSheet visible={showConfirm} onDismiss={() => setShowConfirm(false)}>
  <BottomSheet.Header title="Remove Download" />
  <BottomSheet.Item icon="cloud-download-outline" label="Keep on device" onPress={handleKeep} />
  <BottomSheet.Divider />
  <BottomSheet.Item icon="trash-outline" label="Remove download" onPress={handleRemove} destructive />
</BottomSheet>
```
