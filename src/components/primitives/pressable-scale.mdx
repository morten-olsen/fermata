# PressableScale

<p style={{ color: '#9898A3', fontSize: 16, marginBottom: 24 }}>

## Design rationale

Fermata aims for **satisfying physicality** — interactions should feel tactile, like pressing a real button. `PressableScale` replaces React Native's `TouchableOpacity` with a more refined interaction: instead of fading, it springs down like a physical surface being pressed.

## Usage

```tsx
<PressableScale onPress={handlePress} scaleValue={0.98}>
  <ArtistRow name="Radiohead" />
```
