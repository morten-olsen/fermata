# AlphabetScrubber

<p style={{ color: '#9898A3', fontSize: 16, marginBottom: 24 }}>
  A vertical letter strip for fast-scrolling through alphabetically sorted lists — inspired by the iOS Contacts scrubber and the iPod click wheel.
</p>

<Canvas of={AlphabetScrubberStories.FullAlphabet} />

## Design rationale

When your library has hundreds of artists or thousands of tracks, scrolling one row at a time is too slow. The alphabet scrubber provides the same kind of jump-to-letter navigation that made the iPod's click wheel feel magical: choose a letter and you're there instantly.

### Sparse (only populated letters)

In practice, the scrubber only shows letters that have at least one item. This avoids "dead" letters that scroll to nothing.

## Where it's used

- **Album grid** — jump to albums starting with a letter
- **Artist section list** — jump to the artist section header
- **Track list** — jump to tracks starting with a letter

All three list components compose the scrubber identically: the list provides `letters` (extracted via `extractLetters()`), and the scrubber calls back with the selected letter to trigger `scrollToIndex`/`scrollToOffset`.

    <li><strong style={{ color: '#E8E8ED' }}>Clamped edges</strong> — dragging above the first letter selects A; below the last selects #. No dead zones.</li>
    <li><strong style={{ color: '#E8E8ED' }}>Fires on release</strong> — <code style={{ color: '#D4A0FF' }}>onSelect</code> fires on gesture end, not during. The bubble provides visual feedback during the scrub.</li>
