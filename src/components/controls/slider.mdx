## Design rationale

Fermata's slider is deliberately simple. No labels, no tick marks, no value callouts — just a thin track, a fill, and a thumb. It stays out of the way.

## Where it's used

- **Now Playing** — volume control (fill: `colors.text`)
- **Now Playing** — playback progress (fill: `colors.accent`, via the dedicated FullProgressBar)

## Usage

```tsx
<Slider
  value={volume}            // 0–1
  onValueChange={setVolume}
  fillColor={colors.accent} // optional
  trackColor={colors.surface} // optional
/>
```
