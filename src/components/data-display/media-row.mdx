## Presets

### Track

The default for music. Shows track number, artist subtitle, duration, and optional favourite/download badges. The leading slot automatically switches between track number, equalizer bars (playing), and checkmark (completed).

<Canvas of={MediaRowStories.TrackDefault} />

### Track — playing

When `isPlaying` is true, the track number is replaced by animated equalizer bars and the title turns accent.

<Canvas of={MediaRowStories.TrackPlaying} />

### Track — completed

When `isCompleted` is true, the leading slot shows a checkmark and the title turns muted.

<Canvas of={MediaRowStories.TrackCompleted} />

### Track — with progress

Podcast and audiobook tracks can display a progress bar beneath the title.

<Canvas of={MediaRowStories.TrackWithProgress} />

### Episode

Podcast episodes show an episode number (or mic icon fallback), date subtitle, and remaining duration that accounts for progress.

<Canvas of={MediaRowStories.EpisodeDefault} />

### Episode — playing

<Canvas of={MediaRowStories.EpisodePlaying} />

### Chapter

Audiobook chapters show a chapter number, optional author subtitle, and remaining duration.

<Canvas of={MediaRowStories.ChapterDefault} />

## Custom composition

When presets don't fit, compose directly from sub-components. Wrap children in a `MediaRow` root and pick the pieces you need.

## Layout anatomy

```
┌──────────────────────────────────────────────────┐
│  Leading  │        Content         │  Trailing   │
│  (w-8)    │  Title                 │  badges     │
│  number   │  Subtitle              │  duration   │
│  equalizer│  [Progress]            │  more btn   │
│  check    │                        │             │
└──────────────────────────────────────────────────┘
```

- **Root** — Pressable row with context (isPlaying, isCompleted)
- **Leading** — Fixed-width left slot (32px). SmartLeading auto-picks indicator.
- **Content** — Flex-1 center: title, subtitle, optional progress bar.
- **Trailing** — Right-aligned badges, duration, and action buttons.
