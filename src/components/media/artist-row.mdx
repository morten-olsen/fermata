## Design rationale

Artist rows follow the list row pattern but with a key visual distinction: **circular thumbnails**. This differentiates artists from albums (square) and tracks (no image) at a glance, echoing the convention set by Apple Music and Spotify.

## Where it's used

Artist rows appear in the **Artists tab** of the Library screen, grouped into alphabetical sections by `ArtistSectionList`. Each section gets a header letter and the list includes an `AlphabetScrubber` for fast navigation.
