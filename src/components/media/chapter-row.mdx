## Design rationale

Audiobook chapters are sequential, numbered content. Unlike podcast episodes (date-driven) or music tracks (curated), chapters form a linear progression through a single narrative.

## How it differs from TrackRow

| | TrackRow | ChapterRow |
|---|---|---|
| **Duration** | Total | Remaining (when in progress) |
| **Favourite** | Heart toggle | None |
| **More menu** | "..." button | None |
| **Progress** | Optional bar | Built-in bar |
| **Feel** | Curated tracks | Sequential chapters |
