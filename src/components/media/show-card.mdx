## Design rationale

Podcast shows use square artwork like music albums — podcast cover art is universally square. But shows need different metadata at a glance: **how many episodes** and **are there new ones**.

<div style={{ display: 'grid', gridTemplateColumns: '1fr 1fr', gap: 16, margin: '24px 0' }}>

## How it differs from AlbumCard

| | AlbumCard | ShowCard |
|---|---|---|
| **Content** | Music album | Podcast show |
| **Badge** | Download indicator | Episode count |
| **Indicator** | None | New-episode dot |
| **Fallback icon** | disc | mic |

ShowCard uses the shared `Artwork` primitive, ensuring consistent sizing, fallbacks, and badge positioning across all card types.
